Projeto Preferência de Games

Projeto desenvolvido durante a semana DevSuperior, visando a criação de um aplicativo web para pesquisa de preferencia de games.

A aplicação conta com uma tabela que lista cada novo registro de jogos dos usuários.

E conta também com um gráfico que exibe os jogos, plataformas e gêneros mais populares entre os usuários.

Todas as informações são retiradas do banco de dados PostGreSQL.

Tecnologias usadas:

--BackEnd--

  • Java - SpringBoot
  • PostGreSQL

    • --FrontEnd--

  • TypeScript
  • ReactJS

    • Projeto em produção: sds1-pedro0.netlify.app/
