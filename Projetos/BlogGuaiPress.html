Projeto Blog GuiaPress

Projeto desenvolvido durante a formacao NodeJS, curso ministrado pelo professor Victor Lima na plataforma Udemy.

O projeto permite criar artigos de diversos assuntos e categorias.

O blog conta com um sistema de login, onde só é possível enviar um artigo estando logado.

Quando logado, você pode ver uma lista de todos os artigos e categorias registrados no sistema.

Tecnologias usadas:

  • NodeJS
  • EJS
  • MySQL Server
  • Bootstrap

    • Projeto em produção: Guia Perguntas
