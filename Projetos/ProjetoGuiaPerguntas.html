Projeto Guia Perguntas

Projeto desenvolvido durante a formacao NodeJS, curso ministrado pelo professor Victor Lima na plataforma Udemy.

O projeto de perguntas e respostas no estilo Ask.fm permite que o usuário responda ou realize perguntas de forma completamente anônima.

Todas as informações de perguntas e respostas são armazenadas no banco de dados MySQL.

Tecnologias usadas:

  • NodeJS
  • EJS
  • MySQL Server
  • Bootstrap
